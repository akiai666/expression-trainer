# 🌐 宇宙无敌表达训练系统 - 在线版

**👉 直接使用：[exprtrain.online](https://exprtrain.online)**

无需安装，打开浏览器即用。支持中英双语。

## 功能

- 🎤 **实时语音识别** — 说话即转文字
- ⚠️ **填充词检测** — 自动标红"嗯、啊、那个、就是"
- 🔍 **犹豫词检测** — 标橙"可能、也许、我觉得"
- 📝 **笼统词→精准词** — 实时替换建议
- 🤖 **AI分析报告** — 11个维度详细反馈
- 🌍 **中英双语** — 一键切换，英语练口语同样适用
- 📊 **练习统计** — 时长、字数、填充词占比
